📚 Les Pronoms Relatifs

1. Les pronoms relatifs simples

Les pronoms relatifs sont des mots qui remplacent un nom ou un groupe nominal (appelé l'antécédent) et relient deux phrases. Ils permettent d’éviter les répétitions.

  • Qui → Sujet : Il remplace la personne ou la chose qui fait l'action. Il est toujours suivi d'un verbe.
    Exemple : Le chef qui prépare les plats est célèbre.
  • Que (Qu') → Complément Objet Direct (COD) : Il remplace ce qui subit l’action. Il est toujours suivi d'un nouveau sujet. Attention : "que" devient "qu'" devant une voyelle !
    Exemple : Le plat que j’ai goûté était délicieux. / L'histoire qu'il raconte.
  • Dont → Complément avec "de" : Il remplace un élément introduit par la préposition "de" (parler de, avoir besoin de, avoir envie de, etc.).
    Exemple : Le livre dont je parle est nouveau. (Je parle de ce livre).
  • Où → Lieu ou Temps : Il indique un endroit physique ou un moment précis dans le temps.
    Exemple : Le restaurant où nous dînons. / Le jour où je suis né.
💡 Règle d'or :
• Qui est toujours suivi par un VERBE.
• Que / Qu' est toujours suivi par un SUJET + verbe.

2. Les pronoms neutres (Idées générales)

On utilise ce qui, ce que ou ce dont quand il n'y a pas de nom précis avant. Ils remplacent une idée générale, une phrase entière, ou l'expression "la chose qui / la chose que".

  • Ce qui (Sujet) :
    Exemple : Je ne comprends pas ce qui se passe ici.
  • Ce que / Ce qu' (COD) :
    Exemple : Dis-moi ce que tu veux manger ce soir.
  • Ce dont (Complément avec "de") :
    Exemple : C'est exactement ce dont j'ai besoin ! (J'ai besoin de ça).

3. Les pronoms relatifs composés

On utilise lequel, laquelle, lesquels, lesquelles après une préposition (sur, sous, pour, avec, sans, derrière...). Ils s'accordent en genre (masculin/féminin) et en nombre (singulier/pluriel) avec le nom qu'ils remplacent.

  • Lequel (Masculin Singulier)
  • Laquelle (Féminin Singulier)
  • Lesquels (Masculin Pluriel)
  • Lesquelles (Féminin Pluriel)
Exemple : La table sur laquelle j'ai posé mon sac.
Exemple : Les collègues avec lesquels je travaille sont très sympathiques.
